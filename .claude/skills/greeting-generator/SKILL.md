# 问候语生成器

## 概述

本技能帮助生成各种场景下的中文问候语、祝福语和开场白。适用于邮件、演讲、节日祝福、商务沟通等多种场合。

## 使用场景

### 1. 正式场合
- 商务邮件开场
- 正式演讲开场
- 官方文件开头
- 会议开场白

### 2. 非正式场合
- 日常聊天问候
- 朋友间祝福
- 社交媒体互动
- 个人邮件

### 3. 节日祝福
- 春节、中秋等传统节日
- 生日祝福
- 结婚、升学等庆祝场合
- 新年祝福

### 4. 特殊场合
- 道歉开场
- 感谢表达
- 邀请函开头
- 慰问表达

## 生成原则

### 风格选择
- **正式风格**：使用敬语、规范用语，适合商务和正式场合
- **非正式风格**：亲切自然，适合朋友和日常交流
- **商务风格**：专业简洁，突出效率和尊重
- **节日风格**：温馨祝福，体现节日氛围

### 内容要素
1. **称呼**：根据关系选择合适的称呼（尊敬的、亲爱的、您好等）
2. **时间问候**：根据时间选择合适的问候（早上好、下午好、晚上好）
3. **场合适应**：结合具体场合调整语气和内容
4. **文化适应**：考虑中文文化背景和表达习惯

## 示例

### 商务邮件开场
```
尊敬的[姓名]先生/女士：

您好！感谢您抽出宝贵时间阅读此邮件。
```

### 节日祝福
```
亲爱的[姓名]：

值此[节日名称]佳节，祝您[祝福内容]，愿您[美好祝愿]。
```

### 日常问候
```
[姓名]，你好！

最近怎么样？希望一切顺利。
```
