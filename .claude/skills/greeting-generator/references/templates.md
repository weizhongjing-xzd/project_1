## 商务正式问候

### 邮件开头

**标准格式：**
```
尊敬的[姓名/职位]：

您好！[时间问候]，[简要说明邮件目的]。
```

**示例：**
- 尊敬的张总：您好！感谢您百忙之中阅读此邮件。
- 尊敬的李经理：您好！关于[事项]，我想与您沟通一下。
- 尊敬的客户：您好！很高兴与您建立合作关系。

### 会议开场

**标准格式：**
```
各位[称呼]：

大家好！[时间问候]。今天我们聚在一起，是为了[会议目的]。
```

**示例：**
- 各位同事：大家好！感谢大家参加今天的会议。
- 各位领导：上午好！今天我们讨论[议题]。
- 各位来宾：晚上好！欢迎参加[活动名称]。

## 日常非正式问候

### 朋友问候

**示例：**
- [姓名]，你好！最近怎么样？
- 嗨，[姓名]！好久不见，最近忙什么呢？
- [姓名]，最近还好吗？想你了！

### 社交媒体

**示例：**
- 大家好！今天想和大家分享[内容]。
- 嗨，朋友们！[时间问候]。
- 各位，[时间问候]！今天过得怎么样？

## 节日祝福

### 春节

**示例：**
- 亲爱的[姓名]：值此新春佳节，祝您新年快乐，身体健康，万事如意！
- [姓名]，新年好！祝您在新的一年里工作顺利，家庭幸福！
- 恭祝[姓名]新春快乐，阖家幸福，财源广进！

### 中秋节

**示例：**
- 亲爱的[姓名]：中秋佳节，月圆人团圆。祝您中秋快乐，幸福美满！
- [姓名]，中秋节快乐！愿您和家人团团圆圆，幸福安康！
- 值此中秋佳节，祝[姓名]节日快乐，心想事成！

### 生日祝福

**示例：**
- 亲爱的[姓名]：生日快乐！愿你的每一天都充满阳光和快乐！
- [姓名]，祝你生日快乐！新的一岁，愿你心想事成！
- 生日快乐，[姓名]！愿你永远年轻，永远快乐！

## 特殊场合

### 道歉开场

**示例：**
- 尊敬的[姓名]：非常抱歉给您带来不便。
- [姓名]，对不起，我想为[事项]向您道歉。
- 亲爱的[姓名]：我深表歉意，关于[事项]...

### 感谢表达

**示例：**
- 尊敬的[姓名]：非常感谢您的[帮助/支持]。
- [姓名]，谢谢你的[帮助]，真的很感激！
- 亲爱的[姓名]：我想表达最诚挚的感谢，因为[原因]。

### 邀请函

**示例：**
- 尊敬的[姓名]：诚挚邀请您参加[活动名称]。
- 亲爱的[姓名]：希望您能参加[活动]，期待您的到来！
- [姓名]，你好！想邀请你参加[活动]，有时间吗？

### 慰问表达

**示例：**
- 尊敬的[姓名]：听闻[情况]，深表关切。
- 亲爱的[姓名]：得知[情况]，非常担心，希望你[祝愿]。
- [姓名]，听说[情况]，希望你[祝愿]，保重身体。

## 使用建议

1. **个性化**：根据具体关系和场合调整模板
2. **简洁性**：保持问候语简洁明了，避免冗长
3. **真诚性**：表达要真诚自然，避免过于模板化
4. **文化适应**：考虑中文表达习惯和文化背景
