# 王者荣耀皮肤设计

## 概述

本技能帮助为王者荣耀英雄设计原创皮肤概念方案。提供从英雄分析到完整皮肤设计的全流程指导，包括主题选择、视觉设计、技能特效概念等。

## 设计流程

### 第一步：英雄分析

在开始设计前，需要充分了解目标英雄：

1. **英雄定位**：战士、法师、刺客、射手、辅助、坦克
2. **英雄背景**：官方背景故事、性格特点、世界观设定
3. **技能特点**：主要技能机制、视觉效果特点、战斗风格
4. **现有皮肤**：已有皮肤风格，避免重复，寻找创新点

参考 `references/heroes.md` 获取英雄详细资料。

### 第二步：主题选择

根据英雄特点选择合适的主题方向：

**常见主题类型：**
- **科幻未来**：赛博朋克、机甲、未来战士
- **古风传统**：仙侠、武侠、神话传说
- **现代都市**：时尚、运动、职业
- **幻想魔法**：魔法师、元素、超自然
- **节日限定**：春节、中秋、情人节等
- **IP联动**：动漫、电影、文化IP

**主题选择原则：**
- 与英雄原有设定形成有趣的反差或延续
- 符合英雄的技能特点和战斗风格
- 具有视觉冲击力和辨识度
- 考虑玩家喜好和市场趋势

### 第三步：视觉设计

#### 角色外观设计

**设计要素：**
1. **服装设计**：符合主题的服装风格、材质表现
2. **配色方案**：主色调、辅助色、强调色，考虑游戏内辨识度
3. **武器/道具**：符合主题的武器改造或道具设计
4. **发型与配饰**：发型、头饰、配饰等细节设计
5. **特效元素**：围绕角色的光效、粒子、能量等

**设计原则：**
- 保持英雄核心特征（体型、轮廓、动作）
- 强化主题特色，提升视觉辨识度
- 考虑游戏内小尺寸显示效果
- 平衡美观性与实用性

#### 技能特效设计

为每个技能设计符合主题的特效：

1. **被动技能**：持续效果、触发效果
2. **一技能**：释放特效、轨迹效果、命中效果
3. **二技能**：位移/控制特效、范围指示
4. **大招**：最华丽的特效，符合英雄定位

参考 `references/design-guidelines.md` 了解详细设计规范。

### 第四步：设计文档输出

完成设计后，输出包含以下内容的设计文档：

1. **设计概念说明**：主题、灵感来源、设计理念
2. **角色设计图**：正面、侧面、背面视图（文字描述或概念图）
3. **配色方案**：主色、辅色、特效色
4. **技能特效描述**：每个技能的视觉效果描述
5. **设计亮点**：独特卖点、创新元素

## 设计风格参考

### 史诗皮肤（Epic）
- 主题变化明显，但保持英雄核心特征
- 技能特效有明显变化
- 配色方案完整统一

### 传说皮肤（Legendary）
- 主题完全重塑，可能改变英雄气质
- 技能特效大幅变化，有独特动画
- 可能包含回城特效、待机动作变化

### 限定皮肤（Limited）
- 通常与节日、活动、IP联动相关
- 具有特殊意义和收藏价值
- 设计风格独特，不易复刻

参考 `references/style-guide.md` 了解更多风格细节。

## 设计注意事项

### 必须遵守
- **原创性**：设计必须是原创的，不能抄袭现有皮肤
- **英雄识别度**：保持英雄的核心特征，确保玩家能识别
- **游戏平衡**：特效不能影响游戏平衡性（如遮挡视野）
- **文化尊重**：避免敏感文化元素，尊重不同文化背景

### 设计建议
- **故事性**：为皮肤设计背景故事，增强代入感
- **细节丰富**：在关键部位添加细节，提升品质感
- **动态效果**：考虑角色移动、攻击时的视觉效果
- **玩家反馈**：设计时考虑玩家社区喜好
