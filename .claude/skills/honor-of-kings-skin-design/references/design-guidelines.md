# 王者荣耀皮肤设计规范

本文档提供王者荣耀皮肤设计的技术规范和标准。

## 设计原则

### 1. 英雄识别度
- **核心特征保持**：体型、轮廓、主要动作必须保持
- **关键元素保留**：武器、标志性配饰、技能特征
- **小尺寸识别**：在游戏内小尺寸下仍能清晰识别

### 2. 主题一致性
- **整体风格统一**：服装、武器、特效风格一致
- **配色方案协调**：主色调、辅助色、特效色统一
- **细节呼应**：配饰、纹理、图案相互呼应

### 3. 游戏适配性
- **性能优化**：特效不能过于复杂影响性能
- **视觉清晰**：技能特效不能遮挡视野或误导判断
- **平衡性**：不能通过视觉效果影响游戏平衡

## 品质等级标准

### 勇者皮肤（Elite）
- **主题变化**：轻微主题变化，保持原英雄风格
- **外观变化**：服装、配色有变化，但核心特征不变
- **特效变化**：技能特效有轻微变化，主要是颜色调整
- **价格定位**：288-588点券

### 史诗皮肤（Epic）
- **主题变化**：明显的主题变化，但保持英雄核心特征
- **外观变化**：服装、武器、发型有明显变化
- **特效变化**：技能特效有明显变化，有独特视觉效果
- **语音变化**：可能有新的语音台词
- **价格定位**：788-888点券

### 传说皮肤（Legendary）
- **主题变化**：完全重塑主题，可能改变英雄气质
- **外观变化**：全面重新设计，包括待机动作、回城特效
- **特效变化**：技能特效大幅变化，有独特动画和音效
- **语音变化**：全新的语音包
- **价格定位**：1688点券

### 限定皮肤（Limited）
- **特殊意义**：通常与节日、活动、IP联动相关
- **稀有度**：限时或限渠道获取
- **设计标准**：通常达到史诗或传说品质
- **收藏价值**：具有特殊收藏意义

## 技能特效设计规范

### 被动技能特效
- **持续效果**：围绕角色的光效、粒子、能量
- **触发效果**：触发时的视觉反馈
- **设计要点**：不能过于显眼影响游戏体验

### 一技能特效
- **释放特效**：技能释放时的视觉效果
- **轨迹效果**：技能飞行路径的视觉效果
- **命中效果**：命中目标时的反馈效果
- **设计要点**：清晰表达技能范围和效果

### 二技能特效
- **位移特效**：位移技能的视觉效果
- **控制特效**：控制技能的视觉效果
- **范围指示**：技能范围的清晰标识
- **设计要点**：不影响游戏判断

### 大招特效
- **释放动画**：大招释放时的华丽动画
- **持续效果**：大招持续期间的视觉效果
- **结束效果**：大招结束时的视觉反馈
- **设计要点**：最华丽的特效，符合英雄定位

## 配色规范

### 主色调选择
- **英雄定位**：不同定位的英雄适合不同色调
  - 战士：暖色调（红、橙、金）
  - 法师：冷色调（蓝、紫、白）
  - 刺客：暗色调（黑、紫、深蓝）
  - 射手：亮色调（金、白、浅色）
  - 辅助：柔和色调（粉、绿、浅蓝）
  - 坦克：厚重色调（棕、灰、深色）

### 配色原则
- **对比度**：确保在游戏场景中清晰可见
- **和谐性**：主色、辅色、强调色协调统一
- **主题性**：配色符合皮肤主题
- **识别度**：不同皮肤间有足够区分度

## 材质表现

### 常见材质
- **金属**：武器、盔甲，表现光泽和质感
- **布料**：服装，表现柔软和纹理
- **能量**：特效，表现流动和发光
- **水晶**：装饰，表现透明和折射
- **火焰/冰霜**：元素特效，表现动态和温度

### 材质设计要点
- **符合主题**：材质选择符合皮肤主题
- **层次丰富**：不同材质形成视觉层次
- **细节精致**：关键部位材质表现精细

## 设计文档要求

### 必须包含内容
1. **设计概念说明**：主题、灵感、理念
2. **角色设计描述**：外观、配色、细节
3. **技能特效描述**：各技能视觉效果
4. **设计亮点**：独特卖点和创新

### 推荐包含内容
1. **背景故事**：皮肤的背景设定
2. **设计草图**：关键设计元素
3. **配色方案**：详细的颜色代码
4. **参考素材**：灵感来源

## 常见设计错误

### 避免的问题
1. **过度设计**：特效过于复杂影响性能
2. **识别度低**：改变太多失去英雄特征
3. **主题混乱**：不同元素风格不统一
4. **配色冲突**：颜色搭配不协调
5. **细节缺失**：关键部位缺乏细节

### 设计检查清单
- [ ] 英雄核心特征保持
- [ ] 主题风格统一
- [ ] 配色方案协调
- [ ] 技能特效清晰
- [ ] 细节丰富精致
- [ ] 游戏内识别度高
- [ ] 符合品质等级标准
